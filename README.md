# 行政書士記述Pass プライバシーポリシー

最終更新日: 2026年6月26日

行政書士記述Pass（以下「本アプリ」）は、行政書士試験の記述式学習を支援する独立系の学習アプリです。本アプリは、行政書士試験研究センター、日本行政書士会連合会、総務省、その他試験実施団体が提供、認定、監修するものではありません。

## 取得する情報

本アプリでは、以下の情報を取得または利用する場合があります。

- 学習履歴、回答状況、論点チェック結果、復習対象など、アプリ内の学習データ
- 広告表示に必要な端末情報、広告識別に関連する情報
- アプリの動作、クラッシュ、OS標準の診断情報
- お問い合わせ時にユーザーがメール本文へ記載した情報

本アプリの記述答案は、バージョン1では端末内で字数や論点を確認するために利用します。外部AI API、外部サーバー、法律相談サービスへ送信しません。

## 利用目的

取得した情報は、以下の目的で利用します。

- 学習状況、復習対象、論点チェック結果をアプリ内で表示するため
- 広告を表示するため
- アプリの不具合調査、品質改善、問い合わせ対応のため

## 外部サービス

本アプリでは、以下の外部サービスを利用する場合があります。

- Google Mobile Ads SDK: 広告表示のため
- AppleおよびOS標準の診断機能: クラッシュや動作状況の把握のため

各外部サービスで取り扱われる情報は、各サービス提供者のプライバシーポリシーに従って管理されます。

## 第三者提供

法令に基づく場合を除き、ユーザーの個人情報を本人の同意なく第三者へ提供しません。

## 本アプリで提供しない機能

本アプリは試験学習用の記述練習アプリです。法律相談、個別事案の適法性判断、申請書作成、契約書作成、内容証明作成、行政書士業務支援、AI添削、AI講評、AIによる法律助言、コイン購入、App内課金はバージョン1では提供しません。
